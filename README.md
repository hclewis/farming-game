# Harvest Haven

## Description
This is a simply designed farming game, created as an A Level Computer Science project. The aim of the game is to fulfill the quests in order to achieve the highest score possible, done by planting and harvesting various specified crops.

The game is coded entirely in python, and makes use of the pygame library. All art assets have been created from scratch in ibisPaintX.

## Installation
### Prerequisites
- Python v 3.13.7
- Pygame v 2.6.1

### To Install
- Clone this repo.

  `git clone https://github.com/hclewis/farming-game.git`
- Run `main.py` with python.
